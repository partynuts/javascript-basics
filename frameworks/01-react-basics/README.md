# React 01 | Basics

## Bootstrapping

```
yarn create react-app
```

### Wiring

- `public/index.html`
    - Is served to the browser
    - Contains a #root container
- `src/index.js`
    - Main JS entry point
    - Renders the application and attaches the content to #root
    - Application is loaded from `src/App.js`
- `src/App.js`
